# Screen-Fetch
 Kumpulan Screenfetch [ Logo Distro Linux ]<br>
Apa itu screenfetch? screenfetch adalah tool untuk menampilkan informasi logo distro dan informasi Device.
# How To Install
```
$apt update && upgrade
$pkg install python2
$pkg install git
$git clone https://github.com/MaulanaRyM/Screen-Fetch
$cd Screen-Fetch
$python2 screen.py
```
